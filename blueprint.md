# **App Name**: ARABEX Connect

## Core Features:

- Bilingual Event Hub: A comprehensive homepage displaying key event details, countdown timer, quick registration links, and an event description in both Arabic and English, offering seamless language switching.
- Dynamic Exhibitor & Sponsor Directory: Lists exhibitors with company profiles, booth numbers, product categories, and contact information. Includes search and filter functionalities for easy navigation. Sponsor logos are prominently displayed with links to their websites.
- Unified Registration System: Dedicated forms for visitor and exhibitor registration, ensuring a smooth sign-up process with email confirmation upon successful completion, utilizing Firebase Authentication.
- Event Schedule & Details: Displays the daily agenda of the exhibition, including conferences, workshops, and speaker details in an organized, easy-to-read format.
- Admin Content Management: A secure administrative dashboard for authorized users to add, update, and delete exhibitor profiles, event schedules, and news updates. It also allows managing registrations and viewing basic analytics.
- AI-Powered Description Generator Tool: An admin tool to automatically generate concise summaries or expanded descriptions for exhibitor profiles or news articles based on input keywords or brief outlines, assisting in content creation.
- Venue & Contact Information: Provides essential contact details including email, phone numbers, and social media links. Features an embedded Google Map for venue location and displays a static image of the exhibition floor plan.

## Style Guidelines:

- Light color scheme with a primary deep professional blue (#193A68) for branding and key interactive elements.
- Background color of a very light, almost white tone (#FAFBFC) to ensure clarity and optimal readability for extensive content.
- Accent color of a rich, vibrant gold (#F2BF0D) for highlights, calls to action, and conveying prestige consistent with the ARABEX logo.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-oriented, and strong visual impact reflecting the construction and innovation themes.
- Body text font: 'Inter' (sans-serif) for its high legibility and neutrality, ensuring clear communication across all content, especially with bilingual text.
- Utilize a clean and modern icon set, favoring line icons over filled icons to maintain a professional and uncluttered aesthetic. Icons should be clear and easily recognizable in both Arabic and English contexts.
- Responsive design optimized for mobile, tablet, and desktop. Layouts should emphasize a clear content hierarchy, with ample white space to improve readability and user focus on exhibition information.
- Incorporate subtle and smooth animations for page transitions and interactive elements (e.g., registration buttons, filter toggles). Animations should enhance user experience without causing distractions.
